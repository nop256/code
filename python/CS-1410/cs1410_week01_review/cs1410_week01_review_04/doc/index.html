Exercise: suminout

Description

In this exercise your function will receive three parameters, a list of integers, and two integers. It will add up all values in the list that do not equal either of the two integers.

Function Name

suminout

Parameters

  • nums : a list of integer values
  • a : an integer value
  • b : an integer value

Return Value

A total found by adding all values in nums which do not match a or b.

Examples

suminout([1, 2, 3, 4], 1, 2) -> 7
