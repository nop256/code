Exercise: biggest02
----------------------

### Description

In this exercise, your function will receive two
numeric parameters.  It will return the largest
of the sum of the parameters and the product of
the parameters.

### Function Name

biggest02

### Parameters

* `x` : an integer value
* `y` : an integer value

### Return Value

The sum of `x` and `y` or the product of `x` and `y`, whichever
is larger.

### Examples

    biggest02(2, 3) -> 6
    biggest02(-2, 3) -> 1
