Exercise: Farthest Planet

Description

In this exercise, you will find which of a list of planets is farthest from the center of the star they orbit. For each planet, you have the 3-dimensional coordinates of its location.

A distance can be calculated using the 3D version of the Pythagorean Theorem. In words it is the square root of the sum of the squares of all 3 coordinates.

For example, if a planet is at x = 1, y = 2 and z = 3, its distance is the square root of 1 + 4 + 9.

Function Name

farthest_planet

Parameters

  • planets: A list of planets. Each planet is a dictionary with keys x, y, and z. The values of these keys are positive floating point values. These values represent the planet's coordinates, with the origin being the center of the star they orbit.

Return Value

The planet from planets that is farthest from the center of the star they orbit.

Examples

farthest_planet([{ 'x': 1., 'y': 2., 'z': 3. } , { 'x': 10., 'y': 20., 'z': 30. }]) -> { 'x': 10., 'y': 20., 'z': 30. }

Hints
