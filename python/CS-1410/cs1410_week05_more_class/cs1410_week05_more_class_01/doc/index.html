Exercise: Color Constructor

Description

In this series of exercises, you will create a class to represent a Color. For our purposes, a color is comprised of 3 integer values. They represent how much red, green and blue are used in making the color. The minimum amount for each value is 0, and the maximum amount is 255.

For this exercise, you are only responsible for creating the constructor, and making sure the amounts of red, green and blue are stored in the object.

Files

  • color.py : Python with class definition

Class Name

Color

Constructor

__init__()

Parameters

  • self : the Color object to initialize
  • red : an integer, the amount of red
  • green : an integer, the amount of green
  • blue : an integer, the amount of blue

Action

Initializes the data members to their values from the red, green, and blue parameters.

Examples

c = Color(15, 45, 199) -> c is a Color object with 15 red, 45 green and 199 blue
